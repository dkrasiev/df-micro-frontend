Micro Frontend

Micro frontend
vs
Monolith

Преимущества

  • Независимые релизы
  • Изоляция кода
  • У каждой команды свой полный цикл разработки

Ограничения

  • Общие ресурсы
  • Необходимость оркестрации приложения
  • Коммуникация между приложениями

Имеет смысл, если

  • Большое приложение
  • Несколько команд разработки
  • Нужно переписать приложение
